# STM32RET
> This implementation takes inspiration from ESP32-based firmware: https://github.com/collin80/ESP32RET/tree/master 

STM32F4 Firmware that implements a CAN bus reverse engineering tool that communicates through USB interface and using the GVRET protocol (SavvyCAN-compatible).

GVRET Protocol description: https://github.com/collin80/M2RET/blob/master/CommProtocol.txt

### Requirements:
- STM32CubeMX - https://www.st.com/en/development-tools/stm32cubemx.html
- STM32CubeIDE - https://www.st.com/en/development-tools/stm32cubeide.html
- The project is currently being implemented for an STM32F407VGTx (Discovery board) and it still at the early stages of development
